# Galerie d'Inspiration Créative

Galerie d'Inspiration Créative est un site web moderne dédié aux designers et artistes qui souhaitent explorer, rechercher et sauvegarder des images professionnelles via l'API Unsplash. Le site offre une interface fluide, esthétique et responsive pour une expérience utilisateur immersive.

## Fonctionnalités implémentées

- Recherche intuitive d’images avec suggestions et gestion des erreurs
- Affichage dynamique en grille responsive avec animations et effets hover
- Gestion complète des favoris avec sauvegarde locale (localStorage)
- Visualisation agrandie des images dans une lightbox élégante et accessible
- Interface interactive avec navigation clavier et boutons accessibles

## Technologies utilisées

- HTML5 pour une structure sémantique
- CSS3 modernisé avec Grid, Flexbox, animations et responsive design
- JavaScript ES6+ pour une logique interactive, appels API et gestion du DOM
- API Unsplash pour la récupération d’images haute qualité et libres de droits

## Instructions d'installation et de configuration

1. Cloner le dépôt :
   ```bash
   git clone https://github.com/votre-utilisateur/galerie-inspiration-creative.git
   ```
2. Se positionner dans le dossier :
   ```bash
   cd galerie-inspiration-creative
   ```
3. Créer un fichier `config.js` à la racine avec votre clé API Unsplash :
   ```js
   const API_KEY = 'votre_clef_api_ici';

4. Ouvrir `index.html` dans un navigateur pour utiliser le site.
